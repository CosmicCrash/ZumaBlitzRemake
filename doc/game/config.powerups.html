config/powerups.json

Defines all powerups in the game.

Each powerup has its own unique ID, which is a string. To add a new powerup, simply add an entry to the class with a new unique ID of choice.

Note that powerups are not rendered as sprites, but rather as particles. The particle packet used to render a powerup needs to be persistent.

  • The root entry. Object keys are strings.
    • A single powerup definition.
      • particle: A persistent particle packet which will follow the powerup entity.
      • pickupSound: A sound which will play when the powerup is collected.
      • pickupName: A label which will appear above the shooter when the powerup is collected.
      • pickupFont: A font which will be used for the label.
      • effects: A list of effects to be applied when the powerup is collected. More info below.

Possible powerup effects

Debug information: this can be checked in Level:applyEffect().

replaceSphere

Replaces the current sphere the shooter has with a defined sphere. This can be used to spawn otherwise unobtainable bonus balls, i.e. wild or lightning.

  • A powerup effect.
    • type: replaceSphere
    • color: A sphere ID of a sphere the shooter should get.

multiSphere

Spawns a certain amount of a defined sphere to the shooter. This can be used to i.e. spawn 10 daggers the player can shoot anywhere they want.

  • A powerup effect.
    • type: multiSphere
    • color: A sphere ID of spheres the shooter should get.
    • count: How many of that sphere should be spawned.

speedShot

Replaces the default shooting speed and spawns a speed shot beam effect.

  • A powerup effect.
    • type: speedShot
    • time: The duration of the effect in seconds.
    • speed: A new speed of the shot balls, in pixels per second.

slow

Slows down spheres on the board for a certain amount of time.

  • A powerup effect.
    • type: slow
    • time: The duration of the effect in seconds.

stop

Stops spheres on the board for a certain amount of time.

  • A powerup effect.
    • type: stop
    • time: The duration of the effect in seconds.

reverse

Makes spheres go backwards on the board for a certain amount of time.

  • A powerup effect.
    • type: reverse
    • time: The duration of the effect in seconds.

destroyAllSpheres

Clears the board from all spheres.

  • A powerup effect.
    • type: destroyAllSpheres

destroyColor

Destroys the spheres of a specified color.

  • A powerup effect.
    • type: destroyColor
    • color: The color of spheres to be destroyed.

spawnScorpion

Spawns a scorpion on the board.

  • A powerup effect.
    • type: spawnScorpion

lightningStorm

Launches a lightning storm.

  • A powerup effect.
    • type: lightningStorm
    • count: The amount of strikes.

changeGameSpeed

Changes the game speed to a defined value for a specified period of time.

  • A powerup effect.
    • type: changeGameSpeed
    • speed: The game speed multiplier.
    • duration: A duration of the effect. It is not affected by the game speed itself.
