Overview

This page explains various value types and symbols assigned to it.

  • A number value.
  • A string value.
  • A boolean value (true or false).
  • A list.
  • A JSON object.
  • A 2D vector. It is a table which contains the following values:
    • x: The length of the vector along X axis.
    • y: The length of the vector along Y axis.
  • A color. It is a table which contains the following values:
    • r: The red component, of a value ranging from 0 to 1.
    • g: The green component, of a value ranging from 0 to 1.
    • b: The blue component, of a value ranging from 0 to 1.
  • An expression. There will be a separate page dedicated for it.
  • A pointer which points to a Collectible Generator, in format: "X.json".
  • A pointer which points to a Color Palette, i.e. "images/powerups/wild_pal.png".
  • A pointer which points to a Font, in format: "fonts/X.json".
  • A pointer which points to a particle packet, in format: "particles/X.json".
  • A pointer which points to a Sound Event, in format: "sound_events/X.json".
  • A pointer which points to a Sprite, in format: "sprites/X.json".
