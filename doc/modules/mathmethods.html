Functions

_StrSplit (s, k)
Splits a string `s` with the delimiter being `k`.

Parameters:

  • s string A string to be split.
  • k string A delimiter which determines where to split `s`.

Returns:

    table t A table of strings the input string has been split to.
_StrJoin (t, k)
Combines a table of strings together to produce a string.

Parameters:

  • t table A table of strings to be combined.
  • k string A delimiter which will separate the terms.

Returns:

    string s A combined string.
_StrTrim (s)
Trims whitespace from both the beginning and the end of a given string. Currently supported whitespace characters are `" "` and `"\t"`.

Parameters:

  • s string A string to be truncated.

Returns:

    string s A truncated string.
_StrTrimCom (s)
Trims a line from a trailing comment. The only supported comment marker is `//`.

Example: `"abcdef // ghijkl"` will be truncated to `"abcdef"`.

Parameters:

  • s string A string to be truncated.

Returns:

    string s A truncated string.
