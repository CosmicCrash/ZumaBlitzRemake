getFrameCount ()
Returns the number of frames that should be performed since last call of this function. The internal counter is then decremented by the time equivalent.

The maximum value which can be returned is `MAX_FRAMES`. This means the game will not process more than `MAX_FRAMES` frames at once. In that case, the time counter is still capped to an equivalent of one frame.

Returns:

    number frames The number of frames to be processed.
