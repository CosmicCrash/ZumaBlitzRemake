newSphereColor (omitDangerCheck)
Returns a random sphere color that is on the board. If no spheres are on the board, this function returns the last sphere color that appeared on the board.

Parameters:

  • omitDangerCheck boolean When it is set to true, gets just any random sphere color that is on board provided it is not special (e.g. wild). Setting this to false searches only in colors that are in the danger colors list, and if nothing was found there, it gets a random color from the entire board.

Returns:

    number A sphere color.
colorsMatch (color1, color2)
Returns whether both provided colors can attract or make valid scoring combinations with each other.

Parameters:

  • color1 number The first color to check.
  • color2 number The second color to check.

Returns:

    boolean Whether the check has passed.
usePowerup (data)
Executes a powerup and plays an appropriate sound.

Parameters:

  • data The data of the powerup to use. powerupdata is a quasi-type that conveys following information:
    - name (string) - The name of the powerup, can be one of the following:
    "slow", "stop", "reverse", "wild", "bomb", "lightning", "shotspeed", "colorbomb"
    - color (number) - The powerup color, only exists if name == "colorbomb".
destroyFunction (t, scorePos)
Destroys all spheres on the board if a call of the function f(sphere, spherePos) returns true.

Parameters:

  • t function The function that has to return true in order for a given sphere to be deleted.
  • scorePos Vector2 A position where the score text should be located.

destroyRadius (pos, radius)

destroyVertical (x, width)
